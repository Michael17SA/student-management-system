## CV Bullet

Developed a C# student management console application using OOP principles, input validation, exception handling, list-based storage, student search, removal, and average grade calculation.

## LinkedIn Description

I built a Student Management System in C# to practise object-oriented programming concepts such as abstraction, inheritance, and polymorphism. The application allows users to add student records, validate student IDs, store grades, calculate averages, search records, remove students, and track the total number of students.
